# Проект: Генератор паролей на Python
**Выполнил:**  Пашычкин Максим Станиславович
** Вариант:** Random Password Generator

## О программе
Это простая утилита с графическим интерфейсом. Она помогает быстро создать пароль нужной длины. Можно включать или выключать цифры, буквы и знаки. Все созданные пароли программа запоминает и записывает в файл history.json, чтобы их можно было посмотреть позже.

## Как пользоваться
1. Установите Python.
2. Скачайте файлы проекта в одну папку.
3. Запустите `main.py` через консоль: `python main.py`.
4. Двигайте ползунок, выбирайте настройки и жмите кнопку.

## Что внутри
- Используется библиотека `tkinter` для окошек.
- Данные хранятся в формате `JSON`.
- Генерация через модуль `random`.

## Тесты
- Если не выбрать ни одну галочку, выскочит ошибка.
- Длина пароля ограничена от 4 до 30.
- История корректно загружается при перезапуске.
